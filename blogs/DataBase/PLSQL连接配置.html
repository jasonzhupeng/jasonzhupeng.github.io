    PLSQL连接配置

    jason 3/1/2023 Oracle

    1.解压PLSQL绿色版，目录结构如下

    D:\install\PLSQL
    |-- instantclient_11_2
        |-- tnsnames.ora
    |-- PLSQL Developer
    |-- readme.txt

    2.设置系统环境变量

     NLS_LANG = SIMPLIFIED CHINESE_CHINA.ZHS16GBK
 TNS_ADMIN = D:\install\PLSQL\instantclient_11_2

    3.PLSQL Developer目录下plsqldev.exe发送到桌面快捷方式，双击打开配置环境

    工具 -> 首选项 -> 连接

    Oracle主目录名  = D:\install\PLSQL\instantclient_11_2
OCI库 = D:\install\PLSQL\instantclient_11_2\oci.dll

    4.将编辑好的tnsnames.ora 文件覆盖instantclient_11_2目录下tnsnames.ora，重新打开plsql，配置完成
