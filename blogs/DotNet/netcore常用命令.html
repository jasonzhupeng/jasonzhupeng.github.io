    # ef 数据迁移

    先决条件，ef所需提供程序

    # MS SqlServer

    • install-package Microsoft.EntityFrameworkCore.SqlServer
    • install-package Microsoft.EntityFrameworkCore.Design
    • install-package Microsoft.EntityFrameworkCore.Tools

    # MySql(不支持.net core 3.1类库)

    • install-package MySql.Data.EntityFrameworkCore
    • install-package Microsoft.EntityFrameworkCore.Design
    • install-package Microsoft.EntityFrameworkCore.Tools

    Ef Core CLI

    安装dotnet-ef

    dotnet tool install --global dotnet-ef 安装最新版本ef命令行工具
dotnet tool install --global --version 3.0.0 dotnet-ef 安装指定版本ef命令行工具
dotnet tool update --global --version 3.1.7 dotnet-ef 将ef core升级到某个版本
dotnet tool update -g dotnet-ef 将ef core升级到最新版本

    迁移

    dotnet ef migrations script 20190808074920_Add_SupplierAuditLog 执行指定的迁移日志，生成对应的sql脚本
dotnet ef migrations add InitialCreate 添加迁移
dotnet ef migrations remove 删除最新添加的一个迁移
dotnet ef migrations list 列出迁移
dotnet ef database update 将迁移同步到数据库
dotnet ef database update lastgoodmigrations 将指定迁移同步到数据库，用于回退

    反向工程参数说明

    • -o 实体类输出目录
    • -c ef上下文名称
    • --context-dir ef 上下文输出目录
    • -v 显示详细输出
    • -t 指定生成的表名，多个 -t a -t b ...，不增加此参数则生成生成数据库的全部表结构
    • -f 覆盖
    • --data-annotations 使用数据注解。不增加此参数默认生成fluent api代码
    • --use-database-names 使用数据库列名和表名。不增加此参数默认生成符合C#规则的列名和表名
    • --schema dbo 使用架构。如果使用此参数则生成dbo架构下所有表结构。-t不再起作用
    dotnet ef dbcontext scaffold "Server=127.0.0.1; Database=xx;uid=sa;Pwd=123" Microsoft.EntityFrameworkCore.SqlServer -o Models -c RecruitDbContext --context-dir ./ -t HR_Application --use-database-names -d -v -f
    1

    Nuget CLI

    反向工程参数说明

    • -Tables a,b,c 指定表，如果要抓取数据库所有表则不加此参数
    • -Context 上下文名称
    • -ContextDir 上下文输出目录
    • -OutputDir 实体输出目录
    • -Namespace 命名空间
    • -DataAnnotations 使用数据注解。不增加此参数默认生成fluent api代码
    • -UseDatabaseNames 使用数据库列名和表名。不增加此参数默认生成符合C#规则的列名和表名
    • -Schemas dbo 使用架构。如果使用此参数则生成dbo架构下所有表结构。-Tables不再起作用
    scaffold-dbcontext "Server=172.17.10.210; Database=SOfficeDB;uid=sa;Pwd=1" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Models -ContextDir ./ -DataAnnotations -UseDatabaseNames -Tables sys_Table1,sys_Table2 -V
    1
