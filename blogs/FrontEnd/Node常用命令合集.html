    Node常用命令合集

    jason 1/9/2023 FrontEnd

    🍉 npm，cnpm，yarn 命令合集

    # npm

    npm install nrm -g 用于切换npm镜像
npm install xxx --save -S 安装依赖到生产环境
npm install xxx --dev -D 安装依赖到开发环境
npm install xxx --global -g 安装到全局
npm root -g 查看模块配置
npm config ls -l 查看全局配置
npm install -g cnpm --registry=https://registry.npm.taobao.org    设置镜像库，使用淘宝镜像
npm config set prefix "D:\DevelopFiles\Nodejs\node_global"   配置全局模块安装路径
npm config set cache "D:\DevelopFiles\Nodejs\node_cache"   配置全局缓存路径
npm list -g 查看所有全局安装的包

    # cnpm

    cnpm install npm@latest -g  更新至最新
cnpm install npm@版本号 -g  更新至固定版本

cnpm install -g vue-cli  安装全局vue (2.x)
cnpm install -g vue@cli  安装全局vue (3.x)
cnpm install -g @vue/cli  安装全局vue/cli (4.x)
cnpm install -g @vue/cli-service-global   安装vue sever全局扩展

cnpm uninstall -g vue  卸载全局vue
cnpm uninstall -g vue-cli   卸载全局vue (2.x)
cnpm uninstall -g vue@cli   卸载全局vue (3.x)
cnpm uninstall -g @vue/cli   卸载全局vue (4.x)
